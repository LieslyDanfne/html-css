Paragráfo de quebra linha

Você pode escrever um paragráfo de qualquer jeito.

Se precisar quebrar o texto você precisa ultilizar
a tag < br >. Como exemplo
nesse paragráfo

Vamos adicionar simbolos:
®
©
™
€
£
¥
¢
e varios outros simbolos, so pesquisar o comando.

Vamos adicionar alguns emoji:
🖖
🤪
para adicionar o emoji, tem que colocar &#x(codigo do emoji encntrado na internet).

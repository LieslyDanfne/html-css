Principais formatações

Negrito / Destaque

Nessa formatação temos um termo em Negrito. Para deixar o texto em Negrito basta usar a tag < b >.

Outra opção é a tag < strong > como por exemplo aqui.


Itálico / Ênfase

Nesta frase temos um termo em itálico. Basta usar a tag < i >.

Outra opção é usar a tag < em >. Como netsa palavra.


Para marcar o textos usamos a tag < mark >. Como nesse exemplo aqui.

Texto Deletado

Podemos marcar como deletado para dar ênfase e indicar que foi deletado usando a tag < del >.

Texto Inserdo

Podemos marcar como inserdo para dar ênfase e indicar que foi adicionado usando a tag < ins >.

Outra opção é usar a tag < u > como aqui.


Texto sobrescrito

Para inserir coisas do tipo x20+3. Usando a tag < sup > servindo para elevar um número.


Texto subscrito

Para inserir coisas do tipo H2O. Usando a tag < sub >.
